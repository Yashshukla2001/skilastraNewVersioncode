Skilastra Vite React project (single-file App.jsx)

How to run:
1. Ensure Node.js installed (LTS)
2. cd to project folder
3. npm install
4. npm run dev

Replace constants in src/App.jsx:
- GOOGLE_FORM_WORKSHOP
- GOOGLE_FORM_BATCH
- GOOGLE_FORM_AMBASSADOR
- UPI_ID
- WHATSAPP_NUMBER

Place real assets in public/:
- yash.jpg
- upi-qr.png
- demo-video.mp4 (optional)
- demo-poster.jpg (optional)
